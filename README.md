# vue-pokedex

Desafio Criar uma Pokedex Utilizando Vue.js

Utilizado:
 - Vue 2
 - [PokeApi](https://pokeapi.co/) - Utilizada para consultar os pokémons
 - [Vuesax](https://lusaxweb.github.io/vuesax/) - Utilizado como frontend components

# Demo
